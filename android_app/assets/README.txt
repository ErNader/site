# دایرکتوری assets

این دایرکتوری برای فایل‌های منابع مانند فونت‌ها، تصاویر و سایر فایل‌های استاتیک استفاده می‌شود.

## فونت‌های فارسی

برای استفاده از فونت فارسی در برنامه، باید فایل فونت Vazirmatn را دانلود کرده و در این دایرکتوری قرار دهید.

فونت Vazirmatn را می‌توانید از لینک زیر دانلود کنید:
https://github.com/rastikerdar/vazirmatn/releases

فایل‌های فونت باید با نام زیر در این دایرکتوری قرار گیرند:
- Vazirmatn.ttf (فونت معمولی)

## تصاویر

فایل‌های تصویر مورد نیاز برنامه:
- icon.png (آیکون برنامه، 512x512 پیکسل)
- presplash.png (تصویر صفحه اسپلش، 1024x1024 پیکسل)
